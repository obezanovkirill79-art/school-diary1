## ШАГ 1 — Установи Flutter (один раз)

### Windows:
1. Зайди на **flutter.dev/docs/get-started/install/windows**
2. Скачай Flutter SDK (кнопка "Get the Flutter SDK")
3. Распакуй архив в папку `C:\flutter`
4. Открой **Переменные среды** (Win+S → "переменные среды")
5. В PATH добавь: `C:\flutter\bin`
6. Открой **Android Studio** → скачай с developer.android.com/studio
7. При первом запуске Android Studio: установи Android SDK

### Mac:
```bash
brew install --cask flutter
```

### Проверка что всё работает:
```bash
flutter doctor
```
Должны быть зелёные галочки у Flutter и Android toolchain.

## ШАГ 2 — Создай проект Flutter

Открой терминал (или PowerShell) и выполни:

```bash
# Создать проект
flutter create school_diary

# Зайди в папку
cd school_diary
```

## ШАГ 5 — Запусти на телефоне (для теста)

### Подключи Android-телефон:
1. На телефоне: **Настройки → О телефоне** → тапни 7 раз на "Номер сборки"
2. Появится "Режим разработчика" — включи его
3. В "Параметры разработчика" включи **Отладка по USB**
4. Подключи телефон к компьютеру через USB

```bash
# Проверить что телефон виден
flutter devices

# Запустить приложение на телефоне
flutter run
```

## ШАГ 6 — Собери APK для установки

```bash
# APK для установки (работает на любом Android)
flutter build apk --release

# Файл будет тут:
# build/app/outputs/flutter-apk/app-release.apk
```

Этот файл можно скинуть на телефон и установить!

## ШАГ 7 — Добавить голосовой ввод (speech_to_text)

Пакет `speech_to_text` уже подключён в `pubspec.yaml`.

Для Android нужно добавить разрешение в `AndroidManifest.xml`
(уже сделано в нашем файле):
```xml
<uses-permission android:name="android.permission.RECORD_AUDIO"/>
```

Код голосового ввода добавим в `VoiceOverlay` — попроси меня и я напишу полный код с реальным распознаванием речи!

## ШАГ 8 — Подключить Gemini AI

1. Зайди на **aistudio.google.com**
2. Нажми **Get API key** → Create API key
3. Скопируй ключ
4. В приложении → Настройки → вставь ключ

Всё! ИИ-помощник по ДЗ заработает.

## Частые проблемы

**`flutter doctor` показывает ошибки Android SDK:**
→ Открой Android Studio → SDK Manager → установи Android SDK

**`flutter pub get` не работает:**
→ Проверь интернет-соединение, запусти снова

**Телефон не отображается в `flutter devices`:**
→ Переустанови драйверы ADB, переподключи USB

**APK не устанавливается:**
→ В настройках Android разреши "Установку из неизвестных источников"
